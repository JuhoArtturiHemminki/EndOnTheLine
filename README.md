# EndOnTheLine
EndOnTheLine: An ultra-fast DSP stream processing library in Rust. Leverages parallelized SIMD acceleration and adaptive thresholding for high-throughput voltage/audio data. Achieves up to 8.9 Gelem/s (>35 GB/s) using Rayon and portable_simd, running directly at hardware line speed. Requires Rust Nightly.
